# 物流管理系统 python43

本项目是基于Python语言和Django框架开发的物流管理系统，支持使用MySQL数据库，可根据需求进行定制化开发。

## 项目概述

本系统主要针对物流公司的业务流程进行管理，提供运单、发车、到达、财务以及用户权限等方面的全面管理。

## 技术栈

- Python
- Django
- MySQL
